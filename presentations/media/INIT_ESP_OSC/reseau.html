même langage
Protocole indépendant type de donnée transmise. Protocole tcp/ip est très standard, pas forcément le cas de l'encodage (dante/ravenna)

Langue = protocole (TCP/IP)

protocole ≠ encodage

les paquets de données séparent contenant et contenu, exactement comme une enveloppe et sa lettre

exemple lettre : envoi de musique sous forme de partition ou tablature, même protocole, encodage différent. Système a l'air compatible, les appareils se communiquent mais le message reste incompris (ESTA-sACN/artnet, ravenna/dante...)
adresses
on place une enveloppe dans une enveloppe. 4 couches en TCP, seules 2 nous intéressent

Un appareil ⇔ plusieurs adresses

1 MAC par interface, 1 IP par réseau. Pas de connection = pas d'IP

adresse IP

pour les trouver :
mac/linux : ifconfig
windows : ipconfig \all

l'IP dépend du réseau qu'on considère. IP locale != IP internet
l'IP locale est comme un système de boite postale. Le routeur reçoit des deux côtés et redispatch
Important différence routeur/switchIP statique du même domaine que les autres (ex 192.168.xx.xx), mais différente. IP internet définie par l'ISP, rien qu'on puisse y faire

qui décide de l'adresse IP d'un appareil?

  • IP statique configurée dans l'appareil
  • IP dynamique, assignée par un appareil externe serveur DHCP

les routeurs contiennent un serveur DHCP mais pas les switchs. Les switchs ne connaissent rien à l'IP

soit un serveur DHCP et un client DHCP, soit IP fixe, soit problème
le routeur assigne les IPs
idem, le switch est totalement transparent à l'IP
soit un seul serveur DHCP et les autres en client DHCP, soit IP fixe, soit problème
permet de ne pas taper l'ip dans le navigateur mais un nom. Existe aussi en local sur les routeurs.

Le DNS est un annuaire d'IPs

www.google.com ⇔ 216.58.218.228
www.google.fr ⇔ 172.217.5.227

Chaque appareil reçoit et envoie des milliers de paquets par secondes. Ports sont comme des couloirs entre la carte réseau et les logiciels. Logiciels bloquent les ports qu'ils utilisent

les ports : routent les paquets à l'intérieur de l'appareil

fonctionnent dans les deux sens, peuvent être ouverts ou fermés

souvent notés en face de l'ip : 192.168.0.12:21
